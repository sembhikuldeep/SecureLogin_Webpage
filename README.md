# 🔐 Secure Login Page

A simple and secure login system built using **Python and Flask**.  
This project demonstrates basic authentication, form handling, and web security concepts.

---

## 🚀 Features

- User login authentication
- Password validation
- Secure backend using Flask
- Clean and simple UI
- Beginner-friendly project
- Easy to run locally

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates

## ⚙️ Installation & Setup

1️⃣ Clone the repository


git clone https://github.com/yourusername/secure-login-page.git


2️⃣ Navigate to the project folder


cd secure-login-page


3️⃣ Install Flask


pip install flask


4️⃣ Run the application


python app.py


5️⃣ Open in browser


http://127.0.0.1:5000
