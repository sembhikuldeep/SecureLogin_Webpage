# 🔐 Secure Login Page

A simple and secure login system built using **Python and Flask**.  
This project demonstrates basic authentication, form handling, and web security concepts.

---

## 🚀 Features

- User login authentication
- Password validation
- Secure backend using Flask
- Clean and simple UI
- Beginner-friendly project
- Easy to run locally

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates

---



## 📚 Learning Objectives

This project helps beginners understand:

- Flask web framework
- User authentication basics
- Form handling
- Template rendering
- Simple web security concepts
